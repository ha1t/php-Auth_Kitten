AuthKitten
============================

子猫認証をPHPで実装したものです。
あらかじめ用意された画像の中から、子猫の画像を選択させる事で、
投稿者がSPAMなのか人間なのか判断します。

サポートしているPHPのバージョン
---------------------------

 + 5.2
 + 5.3
 + 5.4

使い方
---------------------------
AuthKittenは、AuthKittenクラスをAuthKittenDrawクラスにわかれています。
